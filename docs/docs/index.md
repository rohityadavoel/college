# DermaEnsemble AI: Multi-Model Dermatology Disease Detection System documentation!

## Description

Developed DermaEnsemble AI, a multi-model dermatology diagnosis system that validates image quality, detects skin regions using MobileNetV3, and predicts diseases using an ensemble of EfficientNetB4, MobileNetV3, and EfficientNetV2-M. The system applies majority voting for reliable classification and retrieves disease details via a MySQL database.
